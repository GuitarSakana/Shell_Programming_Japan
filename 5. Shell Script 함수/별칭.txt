<Shell Script 함수, 별칭>

1. 별칭
    - 여러 명령들을 그룹지어 사용할 수 있음
    - 단순한 명령들의 조합에 새로운 이름을 명명하여 사용

    ex) ls -l이라는 명령어에 별칭을 부여해서 다른 이름으로 사용
    alias lf = ls -l
    unalias lf  <- 별칭 삭제


2. 함수
    - 별칭보다 복잡한 명령조합을 사용하고자 할때 사용
    - 구조
    ex)
        함수명(){
            명령1
            명령2
            ....
        }
