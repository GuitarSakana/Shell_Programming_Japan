<Shell Script 구조>

1. Shell Script 구조(쉘 파일)

    #! 인터프리터 옵션              <- 첫줄은 셔뱅(해쉬뱅)이라고 부르며 스크립트를 처리할 해석기 정의 ex) /bin/bash
    #
    # 주석(Comment)
    
    함수명(){
        명령
        ....
    }
    변수1 = 값
    변수2 = 값

    #주석
    쉘 문법(조건, 선택, 반복 등)
    명령
    ....

    #주석
    함수 호출 ....

=> 꼭 함수 변수가 들어갈 필요 없다/ 한줄에 리눅스 명령만 들어가면 됨



2. 주석의 사용 목적
    - 프로그램 작업 목적
    - 스크립트 개발자 정보
    - 중간중간 기능에 대한 설명



3. 쉘 스크립트를 잘 만드는 방법
    - 시스템 관리 명령을 잘 알기
    - 작업을 위한 명령들과 절차들 알기
    - 작업을 위해 사용할 명령들과 순서들을 알아보기
