<Shell Script 변수>

1. 변수 이름은 대문자로 표기


2. 문자열 변수는 "" 또는 ''로 감싸주면 된다


3. 변수 선언시 띄어쓰기 금지 ex) A1= value <- 공백있으면 안됨 A1=value


4. 변수 호출
    echo $변수 
    echo ${변수} + 문자열/변수    <- {}에 변수를 넣어주면 변수값 + 이후에 문자열 이나 다른 값이 올수 있다.


5. 지역 변수/ 환경 변수
    - 지역 변수: 쉘 프로세스에 국한되어 사용되는 변수.
    - 환경 변수: 자식 쉘 프로세스에 상속되어 사용될 수 있는 변수.


6. 환경 변수 설정 방법
    - export 변수명
    - 위처럼 설정하면 지역변수를 환경변수로 설정 가능


7. 변수 해제
    - unset 변수이름
    - 선언한 변수를 삭제한다
